# 📘 Bachelor Project - Job Search Website

## 📌 Project Name
Job Search Web Application

## 🧾 Description
This is a simple bachelor level web application developed using MERN stack.  
It is a job search platform where users can register, login, and search for available jobs.  
Employers can post jobs and users can apply for jobs easily through the platform.

## 🚀 Features
- 🔐 User Authentication (Login / Register)
- 👤 User & Admin Roles
- 💼 Job Posting (Admin/Employer)
- 🔍 Job Search & Filter
- 📝 Apply for Jobs
- 📊 User Dashboard
- 📱 Responsive Design (Mobile + Desktop)

## 🛠️ Tech Stack

**Frontend:** React.js, HTML, CSS, Bootstrap  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  

## 🎯 Purpose
To help users find jobs easily and allow employers to manage job postings in one platform.
